# NotasMD 📝
> Sistema para pasar de HTML a Markdown-mark y experimentos de lógica con JS.

Este proyecto es una herramienta de aprendizaje y utilidad para convertir estructuras HTML a formato Markdown, además de servir como laboratorio para experimentos de manipulación del DOM y gráficas interactivas.

## 🚀 Características
* **Conversor:** Transforma etiquetas HTML a sintaxis Markdown.
* **Laboratorio JS:** Experimentos de movimiento de objetos en un plano (DOM + CSS).
* **Live Reload:** Entorno de desarrollo con recarga automática.

## 🛠️ Tecnologías
* **Node.js** (Entorno de ejecución)
* **Live-server** (Servidor de desarrollo)
* **Nodemon** (Monitor de cambios)
* **HTML5 / CSS3 / JavaScript (ES Modules)**
